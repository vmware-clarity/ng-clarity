clr-file-list component

The clr-file-list components enables the advanced file input mode. It has no inputs, but you can provide an ng-template with a clr-file-message attribute. That template is rendered for each file and can contain messages via the clr-file-info, clr-file-success, and clr-file-error messages. See the example in the code tab.
