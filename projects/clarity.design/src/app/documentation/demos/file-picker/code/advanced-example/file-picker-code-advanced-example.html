Advanced File Picker Example

This example shows the advanced file picker. This mode will show each selected file via the clr-file-list component. Each file can have its own message text. Disabling the file picker and validation works the same as the examples above. Setting a custom browse button label is also possible as shown above, but the button text won't change based on the selected files.
