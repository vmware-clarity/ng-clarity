States

Enabled

Use when the File Picker component is interactive and ready for input.

Helper message Success message Error message

Active

Use when selected by a user with an input method, such as mouse or keyboard.

Helper message Success message Error message

Disabled

Use when not interactive. For example, when a user does not have permissions to upload a file.

Helper message Success message Error message

Error

Use to communicate a user or system error. This validation can happen when the user moves away from the input or after attempting to submit a form.

Helper message Success message Error message

Success

Use to provide real-time feedback immediately after a user's input. For example, after selecting a file.

Helper message Success message Error message
