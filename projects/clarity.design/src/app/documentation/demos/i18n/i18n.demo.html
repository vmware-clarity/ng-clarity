You can easily translate internal Clarity text into multiple languages.

Internal language strings

Clarity Design System has a list of text strings that it uses internally for items such as icon alt text or button text. When possible, Clarity avoids using text strings that must be translated, and rarely changes the text string list. Any Clarity application that needs to support multiple languages can create different translations and use them for the corresponding languages.

To improve accessibility of the components, Clarity includes a default English title for all icons or non-text interactive elements internal to its components. In order to internationalize them, Clarity includes a ClrCommonStringsService service. This service allows you to provide localized strings for your entire application that will override our default titles.
