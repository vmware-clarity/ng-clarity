Selection Feedback Loop Test

This page tests that [(clrDgSelected)] two-way binding does not create a feedback loop. When you select a row, the (clrDgSelectedChange) output should fire exactly once. If the emit count climbs rapidly or continuously, there is a feedback loop bug.
