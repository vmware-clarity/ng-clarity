Button groups are for creating collections of similar type action buttons.

Usage

Button groups follow normal Button design guidelines. All button options regarding classes and sizes apply to button groups.

Types

Button groups

You can use solid, outline and flat buttons in normal and small sizes.

Solid Group

Solid button groups direct the user's attention to the primary actions the application is suggesting the user take.

Outline Group

Outline groups are used to indicate a set of secondary actions or to reduce the visual noise on a page.

Flat Group

Flat button groups are used as a set of tertiary actions. They are also typically used in Card and above Datagrid components.

Icon button groups

Icon button groups are also available in the solid, outline, and flat types. It’s also best to keep them normal sized (36px). This makes them easier to recognize and to click.

Accessibility recommendation

If you use an icon button but don’t include text, we recommend adding the aria-label="" attribute. This provides screen readers with context for what action your icon button might produce. The text should reflect the action being completed.

Overflow group

Overflow is used when the button group is larger than its containing space, or used to preserve space. The overflow is configurable so you can assign button actions to the dropdown menu.

The overflow button is the last position to the right. Clicking on button will show the overflow dropdown menu.

Overflow with icons

Button groups with text and icons, or just icons, will have text show in the dropdown menu.
Note that text for each action icon is needed for overflow to work properly.

Multiple Groups

Use when you want similar actions to be grouped together and separated from others.

Checkbox groups

Use when a small list of options can be selected from, similar to the Checkbox component.

For this type:

  • Blue indicates an option is selected and white (outline) indicates an unselected option
  • Users can click a button to select, and click again to deselect
  • You can configure the buttons with preselected options

Radio groups

Use when selecting one option from a small list of options, similar to the Radio component.

For this type:

  • Blue indicates an option is selected and white (outline) indicates an unselected option
  • Users can click a button to select, and click again to deselect
  • You can configure the buttons with preselected options

Angular Button Group Component

Basic Structure

Directions

Use the clrMenuPosition input to determine the direction of the Overflow Menu.

CSS Button Group Component

Clarity defines three button group types:

  • Solid. A solid background with light text. These buttons are prominent on the page.
  • Outline. A transparent background with colored border and text. On hover, the button fills with color.
  • Flat. Text in Action Blue, used to indicate an action.

Basic Structure

Types

Mixed
