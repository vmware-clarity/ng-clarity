@clr/addons/card-container renders a drag-and-drop dashboard of resizable cards. Each card hosts an Angular component dynamically, supports layout persistence, and provides a settings panel for toggling card visibility.

Usage

Import AppfxCardContainerModule.

Features

  • Drag-and-drop card reordering
  • Configurable width and height grid units
  • Settings panel to show/hide cards
  • Layout persistence via AppfxContainerPersistenceStore
  • Dynamic card add/remove at runtime
  • Keyboard-accessible drag and drop

Card Container

Three cards in a dashboard. Use the buttons below to add or remove cards dynamically.

AppfxCardContainerComponent Inputs

Input Type Description
cards AppfxCardInternal[] Array of card descriptors (required)
containerId string Unique ID used for layout persistence (required)
showCardContainerSettings boolean Show settings panel; default true
dragDropEnabled boolean Enable drag-and-drop; default true
persistenceStore AppfxContainerPersistenceStore Custom persistence adapter (optional)

AppfxCard Interface

Property Type Description
id string Unique card identifier
title string Card header title
componentClass Type<any> Angular component rendered in the card body
unitWidth number Card width in grid units
unitHeight number Card height in grid units
canHide boolean If false, the card cannot be hidden from the settings panel; default true
context AppfxCardContext Key/value pairs passed as input properties to the dynamically created componentClass instance
cardZoomSizes AppfxCardZoomSizes Overrides for unitWidth/unitHeight at 2x and 4x zoom levels
