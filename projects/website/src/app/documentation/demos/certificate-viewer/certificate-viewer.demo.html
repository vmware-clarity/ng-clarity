Certificate Viewer

The certificate viewer (appfx-certificate-viewer) displays detailed X.509 certificate chain information. It renders a General tab (validity dates, subject, issuer, fingerprints) and a Details tab with the fully parsed certificate tree.

General tab

The General tab contains a short summary of the leaf certificate:

  • Issued To — the subject to which the certificate was issued.
  • Issued By — the issuer of the certificate.
  • Validity — the certificate validity period.
  • Fingerprints — the SHA-1 and SHA-256 fingerprints of the certificate.

Details tab

The Details tab is an explorer view of the provided certificate chain with three sections:

  • The Certificate Hierarchy tree represents the certificate chain beginning with the root certificate. Selecting a certificate in the hierarchy tree populates the Certificate Fields tree.
  • The Certificate Fields tree contains all of the fields of the selected certificate. The common fields are translated and the rest are presented as PKI OIDs. Selecting a field in this tree displays its value in the Field Value area.
  • The Field Value of the selected field is a monospace font area. Common certificate fields are presented in a human-readable format; otherwise the value is formatted as a hex string.
