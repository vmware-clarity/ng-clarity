Compact Datagrid

One way to increase the information density of your Datagrid or to decrease the amount of space it takes up, add the datagrid-compact class to it. This class decreases the amount of whitespace paddings in the default Datagrid style.

User ID Name Pokemon @for (user of users; track user; let i = $index) { {{ user.id }} {{ user.name }} {{ user.pokemon.name }} } {{ users.length }} users

Overflow ellipsis

Another way to increase the information density of your Datagrid is to add datagrid-overflow-ellipsis class to it. This class will truncate the visible data in the default Datagrid style so that you can display more rows in the fixed height.

User ID Name Long Text Static Width 300px Pokemon Creation date @for (user of users; track user; let i = $index) { {{ user.id }} {{ user.name }} {{ longString }} {{ user.pokemon.name }} {{ user.creation | date }} } {{ users.length }} users

Full example

  • datagrid-compact will reduce datagrid cell vertical padding.
  • datagrid-overflow-ellipsis will set datagrid cell overflow to ellipsis and will truncate the visible data. Because this class is visually limiting please consider one of the following examples below.
    1. clr-tooltip component in the beginning of the cell.
    2. clr-dg-row-detail with clrDgReplace set to true. Use this method if there is no contradiction with your current implementation of clr-dg-row-detail.
    3. title HTML attribute (is A11Y contrast limited). Not recommended.
