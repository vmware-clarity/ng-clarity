Disable Select All

The clrDgSelectAllDisabled input disables the "select all" checkbox in the datagrid header, preventing users from selecting or deselecting all rows at once. Individual row checkboxes remain fully functional.

When to use

Disabling the "select all" checkbox is useful in the following scenarios:

  • Loading state — While data is being fetched from the server, disable the checkbox to prevent users from interacting with the selection before the data is ready.
  • Restricted bulk operations — When your application has business rules that prohibit bulk selection (e.g., the user does not have permission to perform bulk actions, or the dataset is too large for bulk selection).
  • Conditional availability — When the ability to select all rows depends on external state, such as a workflow step or a feature flag.

Basic usage

Set [clrDgSelectAllDisabled] to true on the <clr-datagrid> component to disable the "select all" checkbox. The default value is false.
