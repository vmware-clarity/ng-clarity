The cds-divider attribute styles an <hr> element as a Clarity divider that inherits theme colors and spacing. It supports both horizontal and vertical orientations and can be customized with CSS custom properties.

Usage

Add the cds-divider attribute to a native <hr> element. No module import is needed — the styles come from the global Clarity stylesheet.

  • Use a horizontal divider to separate sections of content stacked vertically.
  • Use a vertical divider to separate items arranged side by side (toolbars, action groups).
  • Place dividers inside a cds-layout container so they stretch to fill the available space as expected.

Types

Horizontal

The default orientation. Spans the full width of its container.

Use dividers inside layouts so they span the dimensions of their containers as expected.

This allows them to span the dimensions of their containers as expected.

Vertical

Set orientation="vertical" for a vertical divider. Use inside a horizontal cds-layout container so it stretches to the container's height automatically.

Customization

Override the divider's color and thickness using the --color and --size CSS custom properties.

Default

Blue, 2px thick

Accessibility

The native <hr> element has an implicit role="separator". No additional ARIA attributes are needed in most cases. If the divider is purely decorative and should be hidden from assistive technology, add aria-hidden="true".
