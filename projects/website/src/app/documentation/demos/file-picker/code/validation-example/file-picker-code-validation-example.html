Validation Example

Several validations are supported out of the box:

  • Required: Use the native required attribute on the file input element. The validation will pass if at least one file is selected.
  • File type: Use the native accept attribute on the file input element. It is recommended to include both file extensions and MIME types for the best browser and OS support. The validation will pass if all selected files match an accepted file extension or MIME type.
  • File size: Use the [clrMinFileSize] and [clrMaxFileSize] inputs on the file input element. The validation will pass if all selected files are within the given range (measured in bytes). If only the min or the max file size is specified, the allowed file size range will be open ended.
