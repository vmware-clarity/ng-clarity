Nav Groups

The clr-vertical-nav-group component should be used to create nav groups. The links inside of a nav group should be wrapped inside of a clr-vertical-nav-group-children component.

Mixed Navigation

Navigation Links can also be added directly inside of the clr-vertical-nav without creating a clr-vertical-nav-group to create a mixed navigation.
