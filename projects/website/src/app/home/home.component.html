New: Version 18.0.0 Release

Dropped @cds/core Dependency

Icons, CSS tokens, and layout utilities (cds-icon, cds-layout, cds-text) are now shipped directly inside @clr/angular and @clr/ui, eliminating the external @cds/core dependency and simplifying installation.

Clarity Addons

A new @clr/angular/addons package provides additional components and utilities that complement the core library while keeping the main bundle lean.

Component Improvements

New Angular Badge and Label components with outline and solid variants, an editable Combobox state, Datagrid unsort capability, simplified Datagrid selection logic, and decoupled Accordion & Stepper via a shared CollapsiblePanel abstraction.

Why Update to v18?

Angular 21 & CDK Popovers

Clarity 18 upgrades to Angular 21 and introduces secondary entry points for all modules, enabling fine-grained tree-shakable imports. All popover-based components — Dropdown, Signpost, Tooltip, and Datagrid overlays — now use Angular CDK overlays for positioning, with a new point-based origin API for advanced placement scenarios.

Smaller Bundles & Faster Builds

Secondary entry points and the removal of @cds/core reduce bundle sizes, while dropping IE 11 and legacy Edge workarounds removes dead code and simplifies the CSS output.
